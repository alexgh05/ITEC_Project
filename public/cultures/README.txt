Cultures Directory

This directory contains images for each culture featured on the CultureDrop website.

Required images:
- tokyo.jpg: Image representing Tokyo culture
- newyork.jpg: Image representing New York culture
- lagos.jpg: Image representing Lagos culture
- seoul.jpg: Image representing Seoul culture
- london.jpg: Image representing London culture
- berlin.jpg: Image representing Berlin culture

Image requirements:
- Format: JPG/JPEG
- Aspect ratio: 4:3 (recommended)
- Recommended size: 800x600px
- High-quality images that represent the essence of each urban culture

To update images, simply replace the files in this directory while keeping the same filenames.
